# Обновление сайта за 2 минуты

## Где править
- Основной файл: `index.html`

## Локальная проверка
- Открыть `index.html` в браузере
- Проверить: меню, модалки, отзывы, почту

## Как заливать на хостинг
1. Открыть панель хостинга или FTP/SFTP
2. Перейти в корень сайта (`public_html` или `www`)
3. Заменить файл `index.html` на новый
4. Проверить сайт по `https://ваш-домен`

## Быстрый чек после обновления
- Открывается главная страница
- Клики по "О нас / Услуги / Контакты" работают
- В отзывах точки переключают слайды
- Email ссылка открывает форму письма

## Откат (если что-то сломалось)
- Взять предыдущую версию `index.html` из Git и перезалить
